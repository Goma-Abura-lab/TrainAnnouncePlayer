###### 1　はじめに ######
TrainAnnouncePlayer（通称「TAP」）（以下「本アプリ」といいます）をダウンロードしていただき、ありがとうございます。
ご使用になる前に、必ず本ReadMeをご一読ください。

###### 2　本アプリの特徴 ######
本アプリは、さまざまな鉄道シミュレーターにおいて「車内から聞こえる放送」を再生することに特化した、Windows版のMP3プレイヤーです。
Mac、Linuxでは動作しません。
※実行時にSmartscreenの警告が出ることがあります。ウイルスは含まれていませんが自己責任で実行してください。

本アプリでは、主に以下の放送の再生を想定しています。
・車内放送（次駅案内放送、到着予告放送）
・駅名連呼（「○○、○○、ご乗車ありがとうございます」といった放送）
・駅メロ・戸閉放送

特定のキーを押すことで、ゲームをプレイ中であっても（つまり本アプリがアクティブになっていない場合でも）、音声の再生・一時停止・停止・スキップ・リセットなどの操作を行うことができます。

本アプリには、東海道線普通列車 熱海行き（東京～品川間）のシナリオおよび音声ファイルが同梱されています。

また、本アプリはユーザーの皆様が、お好みの路線や放送を自由に追加することを想定して設計されています（「3　アドオン制作について」を参照してください）。

###### 3　アドオン制作について ######
本アプリは、JSONファイルでシナリオを作成し、そのJSONファイルが指定した音声ファイル（mp3ファイル）を再生する仕組みとなっています。
そのため、音声ファイルとJSONファイルを追加することで、簡単にアドオンを制作することができます。ぜひ、お好みの路線や放送を追加してお楽しみください。

しかし、JSONファイルを手作業で作成するのは手間のかかる作業です。そこで、本アプリ用のJSONファイルを作成するための「TrainAnnouncePlayer_Editor」を同梱しました。
これにより、コーディングの知識がない方でも簡単にシナリオを作成することができます。

アドオン制作について詳しくは「TrainAnnouncePlayer_Editor使用マニュアル」をご覧ください。

また、さまざまな駅で使用されている一般的な放送の音声ファイル集などのアドオンの公開も歓迎します。

制作したアドオン（シナリオ・音声ファイル集など）を公開する場合、開発者への連絡は一切不要です。なお、音声ファイルに関する著作権には十分注意してください。
